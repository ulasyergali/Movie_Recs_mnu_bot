# Movie Recommendation Bot

🎬 Telegram bot that recommends movies based on genres, manages Favorites and Watched movies, and allows rating watched movies from 1 to 5 stars.

## Features

- Choose a movie genre and get 5 random movie recommendations
- Add movies to Favorites
- View and manage your Favorites
- Move movies to Watched list
- Rate watched movies from 1 to 5 stars
- Buttons for all actions (no typing needed)

## Requirements

- Python 3.11
- python-telegram-bot==20.4
